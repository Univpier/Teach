# Урок 3. Публикация на GitHub Pages

## Как работает деплой

Workflow в `.github/workflows/deploy.yml` автоматически:

1. Устанавливает Python и зависимости
2. Выполняет `mkdocs build`
3. Публикует артефакт в GitHub Pages

## Что проверить

- В репозитории включен `Settings -> Pages -> Source: GitHub Actions`
- Изменения запушены в ветку `main`
- Вкладка `Actions` показывает успешный run

## Практика

Сделайте небольшой коммит в документацию и убедитесь, что сайт обновился.
