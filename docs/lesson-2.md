# Урок 2. Структура проекта

## Рекомендуемая структура

```text
docs/
  index.md
  lesson-1.md
  lesson-2.md
  lesson-3.md
  assets/
    images/
    stylesheets/
```

## Зачем это нужно

- `docs/assets/images/` для логотипов и иллюстраций
- `docs/assets/stylesheets/` для кастомизации темы
- Разделение уроков по файлам упрощает сопровождение
